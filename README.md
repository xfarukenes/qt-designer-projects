Qt Designer Projects

A collection of desktop GUI applications built with Python and Qt Designer.

## 📌 About

This repository contains various desktop applications developed using Python and Qt Designer.  
Each project focuses on building user-friendly graphical interfaces and improving GUI development skills.

## 🛠 Technologies Used

- Python
- PyQt / PySide
- Qt Designer

## 📂 Project Structure

Each project is organized in its own folder and includes:

- `.ui` files (Qt Designer interface files)
- Python source files
- Required resources (if any)
